ChordController documentation
===========================================

.. toctree::
   :maxdepth: 2
   :hidden:

ChordController is a program and software library for using game controllers as MIDI musical controllers. Although optimized for Xbox controllers by default, its rich configuration system allows you turn a wide variety of game peripherals into customized musical instruments.
